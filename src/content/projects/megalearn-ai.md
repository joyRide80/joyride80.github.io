## Role & Strategy

As Co-founder and AI Product Designer, I designed and built the entire system using an AI Agent architecture. This project represents a shift in my career toward AI-native design, where I moved beyond traditional UI to build internal AI tools that automate game production and content generation.

- **AI-Generated Assets**: I developed a pipeline where 3D voxel models are generated from text prompts, allowing us to scale the game world infinitely with a minimal workforce.

![|span:3|col:2](/images/projects/megalearn-ai/cuteVoxel.mp4)

- **Procedural World Building**: I designed a modular terrain system that generates game environments based on parameters like humidity and temperature.

![|span:2|col:1](/images/projects/megalearn-ai/terrainEditor-shape.png)

![|span:1|col:3](/images/projects/megalearn-ai/terrainEditor-climate.png)

![|span:1|col:4](/images/projects/megalearn-ai/terrainEditor-surface.png)

- **Rapid Prototyping**: Leveraging my background in game design, I built real-code prototypes to test complex gameplay loops—Hypothesis, Experiment, and Conclusion—iterating directly with teachers and parents to refine the MVP.

![|span:1|col:1](/images/projects/megalearn-ai/mission-overlay.png)

![|span:2|col:2](/images/projects/megalearn-ai/writing.png)

- **AI-Driven Emotional Narrative**: I designed the conversation scenes to use AI-generated emotional tags assigned to each interaction. This allows the system to dynamically adjust character expressions and the game's atmosphere based on the emotional tone of the dialogue, creating a more empathetic and immersive learning experience.
